# TinyPBR
This project is a research project for myself to learn about PBR.
The goal for now is a small renderer, able to present a few models using PBR as their pipeline. 

### API/Libraries/Tools used for the project:
- GLFW
- GLAD
- ImGui
- TinyglTF
